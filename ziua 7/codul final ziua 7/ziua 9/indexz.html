Checklist WAI-ARIA

Criteriu Status Detaliu
Unic <h1> în pagină ✓ OK Un singur H1 – "Accessibility Audit & WAI-ARIA"
Ierarhie H1 → H2 → H3 ✓ OK Nicio secvență sărită
aria-label pe butoane icon-only ✓ OK Editează, Caută, Setări, Share, Detalii
aria-label pe link-uri social ✓ OK Instagram, X, LinkedIn, GitHub, YouTube
Iconițe decorative cu aria-hidden="true" ✓ OK Toate <i class="fa*">
Landmark roles explicite ✓ OK role="navigation", role="main", role="contentinfo"
Imagini informative cu alt ✓ OK Imagini decorative cu alt=""
Skip-link pentru utilizatori keyboard ✓ OK "Sari la conținut principal" – vizibil la focus
:focus-visible stilizat ✓ OK Box-shadow albastru pe toate elementele interactive
Modal cu role="dialog" & focus trap ✓ OK aria-modal, aria-labelledby, aria-describedby
Buton hamburger cu aria-expanded ✓ OK Actualizat dinamic la toggle
Live region pentru notificări ✓ OK aria-live="polite" în footer
